Machine Learing.

Machine learning is an application of artificial intelligence (AI) that provides systems the ability to
automatically learn and improve from experience without being explicitly programmed. Machine learning
focuses on the development of computer programs that can access data and use it learn for themselves.

In this project I have tried predict the species of IRIS Flower,by using knn Algorithm and IRIS Flower dataset.
have also plotted the accuracy score of prediction of model using matplotlib.
