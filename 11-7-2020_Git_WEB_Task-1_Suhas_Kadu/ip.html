Image Processing
Image processing is a method to perform some operations on an image, in order to get an enhanced image
or to extract some useful information from it. It is a type of signal processing in which input is an
image and output may be image or characteristics/features associated with that image.Nowadays, image
processing is among rapidly growing technologies.


In this Project I have OpenCV.When user clicks anywhere on the image he gets the color and RGB value of that
pixel.This project is base for many image drawing and image processing applications.
